# 后端无状态API

## Tag

### [get] /api/tags

	Parameters:
		- [boolean]onlyMine: [Filter]是否仅返回当前用户持有的标签

	Response:
		- 符合筛选要求的标签列表
		- <Sample>:
		[
			<Tag Model 1>,
			<Tag Model 2>
		]

### [get] /api/tags/{tagId}

	Parameters:
		- [long]tagId: [Filter]tag的id

	Response:
		- 符合筛选要求的标签对象
		- <Sample>:
		<Tag Model>

### [get] /api/tags/{tagContent}

	Parameters
		- [string]tagContent: [Filter]tag的content域

	Response:
		- 符合筛选要求的标签对象
		- <Sample>:
		<Tag Model>
